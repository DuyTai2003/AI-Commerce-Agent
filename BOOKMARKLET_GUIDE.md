# 🚀 1688 Data Feeder - Hướng dẫn cài đặt

## Cách 1: Bookmarklet (Khuyên dùng - 0 đồng)

### Bước 1: Tạo bookmark
1. Mở Chrome, nhấn `Ctrl+Shift+B` để hiện thanh bookmark
2. Chuột phải vào thanh bookmark → **Add page...**
3. Đặt tên: `1688 Feeder`
4. Copy TOÀN BỘ nội dung file `public/bookmarklet.js` vào ô URL
5. Nhấn **Save**

### Bước 2: Sử dụng
1. Mở trang sản phẩm bất kỳ trên **1688.com**
2. Click vào bookmarklet `1688 Feeder` trên thanh bookmark
3. Popup hiện ra → xác nhận dữ liệu → nhấn OK
4. Tự động chuyển sang AI Commerce Agent với dữ liệu đã điền sẵn!

### Mẹo:
- Bookmarklet hoạt động trên mọi trình duyệt (Chrome, Edge, Firefox, Brave)
- Nếu không quét được, hãy copy-paste thủ công vào app
- Bookmarklet KHÔNG gửi dữ liệu về bất kỳ server nào — chỉ chạy trên trình duyệt của bạn
